# Taller 2 - Django Almacenamiento de Libros

Aplicación Django con dos modelos relacionales (Autor y Libro) para la gestión de una biblioteca.

## Equipo
- Integrante A: Módulo Autor (`feature/autor`)
- Integrante B: Módulo Libro (`feature/libro`)

## Requisitos
- Python 3.x
- Django

## Instalación
```bash
python -m venv venv
source venv/bin/activate
pip install django
```

## Ejecución
```bash
cd proyecto
python manage.py migrate
python manage.py runserver
```
